Deskirpsi Proyek:
Sistem pemantauan otomatis untuk ruang belajar seperti perpustakaan atau ruang kelas yang melakukan pemantauan terhadap kebisingan dan kualitas udara secara real-time untuk memastikan suasana belajar yang nyaman dan kondusif. Sistem menggunakan dua Arduino, yaitu Master dan Slave. Slave dihubungkan ke dua sensor analog, yaitu sensor suara (KY-037) untuk mendeteksi tingkat kebisingan dan sensor gas (MQ-2/MQ-135) untuk memantau kualitas udara. Data pemantauan dapat dikirim dari Slave ke Master menggunakan salah satu protokol komunikasi (I2C atau SPI). Setelah Master mendapatkan data ia dapat menampilkannya melalui Serial Monitor. Selain itu, Ketika tingkat kebisingan dan/atau konsentrasi gas melampaui ambang batas yang ditentukan, Master akan memberikan peringatan langsung melalui LED warna-warni dan bunyi buzzer. Selain itu, adapula button untuk mematikan bunyi buzzer sebagai input dari user supaya ia tidak berbunyi terus menerus setelah mendeteksi tingkat suara/gas yang melewati ambang batas.

Implementasi Modul:

Modul 2 (Introduction to Assembly & I/O Programming):
- Implementasi program Assembly untuk mengontrol komponen I/O pada Master seperti LED warna-warni dan buzzer untuk indikator peringatan
- Implementasi program Assembly untuk mengontrol button sebagai input user untuk mematikan buzzer
- Pengaturan pin digital untuk komponen I/O dasar (LED, buzzer, button)

Modul 3 (ADC - Analog to Digital Conversion):
- Implementasi konversi analog ke digital untuk sensor suara (KY-037) pada Slave 
- Implementasi konversi analog ke digital untuk sensor gas (MQ-2/MQ-135) pada Slave
- Pemrosesan data analog menjadi nilai digital yang merepresentasikan tingkat kebisingan dan kualitas udara
- Penentuan ambang batas (threshold) untuk tingkat kebisingan dan konsentrasi gas

Modul 4 (Serial Port - USART):
- Implementasi komunikasi serial pada Master untuk menampilkan data pemantauan ke Serial Monitor
- Pengiriman data hasil pemantauan secara real-time dari Master ke komputer
- Format pesan untuk menunjukkan status kebisingan dan kualitas udara beserta nilai masing-masing

Modul 6 (Timer):
- Implementasi timer untuk pengambilan data sensor secara berkala pada Slave
- Pengaturan interval pengambilan data suara dan gas pada interval waktu yang tetap
- Implementasi timer untuk mengatur frekuensi kedipan LED peringatan pada Master
- Pengaturan durasi bunyi buzzer saat peringatan diaktifkan

Modul 7 (Interrupt):
- Implementasi interrupt untuk button yang mematikan bunyi buzzer
- Penanganan interrupt pada kondisi threshold terlampaui pada data yang diterima Master
- Prioritas interrupt untuk kondisi darurat saat kualitas udara sangat buruk

Modul 8 (I2C / SPI):
- Implementasi protokol komunikasi I2C atau SPI untuk pengiriman data dari Slave ke Master
- Konfigurasi Slave sebagai pengirim data sensor dan Master sebagai penerima
- Format paket data untuk transmisi nilai kebisingan dan kualitas udara
- Penanganan error komunikasi antara Master dan Slave

Modul 9 (Sensor Interfacing):
- Implementasi antarmuka untuk sensor suara KY-037 untuk pembacaan tingkat kebisingan
- Implementasi antarmuka untuk sensor gas MQ-2/MQ-135 untuk pemantauan kualitas udara
- Kalibrasi sensor untuk mendapatkan nilai pembacaan yang akurat
- Pengolahan sinyal sensor untuk mengurangi noise dan fluktuasi pembacaan
